 2. Git-Hoster aufrufen, z. B. Gitlab
 3. User anlegen bzw. einloggen, falls User schon vorhanden ist.

 ## Setup

 ```
    git config --global user.email <Ihre-Email>
    git config --global user.name <Ihr-Name>
 ```
